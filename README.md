# Calc WinForms RadioButton

Калькулятор на C# Windows Forms с выбором операции через RadioButton.

<img width="739" height="828" alt="calc-winforms-radiobutton" src="https://github.com/user-attachments/assets/fc1cfad5-9f03-4661-88a8-7b0cbfc9b432" />

## Функциональность

- Ввод двух чисел
- Выбор операции через RadioButton:
  - Сложение (+)
  - Вычитание (−)
  - Умножение (×)
  - Деление (÷)
- Сложение выбрано по умолчанию при запуске
- Вычисление результата по кнопке
- Окно открывается по центру экрана
- Кнопка закрытия программы

## Интерфейс

- textBox1 — первое число
- textBox2 — второе число
- radioButton1 — сложение
- radioButton2 — вычитание
- radioButton3 — умножение
- radioButton4 — деление
- textBox3 — результат
- button1 — вычислить
- button2 — выход

## Пример работы

**Сложение (по умолчанию):**
Ввод: 15, 7 → результат 22

**Деление:**
Выбор деления, ввод: 20, 4 → результат 5

## Куда можно встроить

- В инженерные калькуляторы
- В финансовые приложения
- В образовательные программы
- В системы автоматизации расчётов

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При делении на ноль программа вызовет исключение — для production рекомендуется добавить проверку. RadioButton1 установлен как Checked = true в конструкторе.
